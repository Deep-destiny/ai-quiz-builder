## üìù Overview
**AI Quiz Builder** is an intelligent web application that automates the creation of educational assessments. Instead of manually writing questions, users can provide any text (articles, book chapters, or notes), and the system uses **Natural Language Processing (NLP)** to generate relevant Multiple Choice Questions (MCQs) and descriptive queries instantly.

This project focuses on the intersection of **EdTech and Generative AI**, providing a tool for educators and students to test knowledge with zero manual overhead.

## ‚ú® Key Features
* **Automated Question Generation:** Uses NLP models to identify key concepts and entities in a text to form contextually accurate questions.
* **Instant Evaluation & Feedback:** Features a real-time scoring engine that evaluates user responses and provides detailed explanations for correct/incorrect answers.
* **Dynamic Content Extraction:** Capable of parsing various input formats (PDF/Text) to create diverse quiz sets.
* **Responsive Dashboard:** A clean, intuitive UI built for both students (taking quizzes) and creators (generating quizzes).
